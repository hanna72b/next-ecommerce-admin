# Commerce Admin Panel

A modern and responsive e-commerce admin dashboard built with Next.js, TypeScript, Tailwind CSS, and shadcn/ui.

## Live Demo

🔗 https://commerce-admin-panel.netlify.app/

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- shadcn/ui
- React Hook Form
- Zod

## Features

- OTP Authentication UI
- Responsive Dashboard
- User Management
- Dark / Light Theme
- Modern UI Components
